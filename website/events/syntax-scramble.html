About the Event

Syntax Scramble is an intense blind coding challenge where participants write code without executing it, then debug another participant's code without knowing the original problem statement. Test your coding skills, logical thinking, and debugging abilities in this unique competition format.

Competition Rounds

  • Round 1: Blind Coding (45 min)
  • Round 2: Debugging (25 min)
  • Round 3: Execution & Judging (20 min)
