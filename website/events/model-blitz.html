About the Event

Model Blitz is an exciting machine learning competition where teams compete to build the most accurate predictive models. Participants will work with real-world datasets, applying their ML expertise to solve challenging problems in a time-constrained environment.

Competition Format

  • Dataset Analysis Round
  • Feature Engineering Challenge
  • Model Development
  • Performance Optimization
  • Final Presentation

Rules & Guidelines

  • Teams must have 2-3 members
  • Bring your own laptops with required packages installed
  • Internet access will be provided
  • Pre-trained models are not allowed
  • Valid college ID mandatory
  • Judges' decision is final

Required Tools

  • Python 3.8+
  • TensorFlow/PyTorch
  • Scikit-learn
  • Pandas & NumPy

Judging Criteria

  • Model Accuracy (40%)
  • Computational Efficiency (20%)
  • Code Quality (20%)
  • Presentation (20%)
